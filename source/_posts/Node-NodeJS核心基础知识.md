### 关于 Node 和 Javascript

* JavaScript 是一门编程语言，通常在浏览器的宿主环境中运行，网页是它的主战场，用户点击反馈/数据异步加载/动画控制等需要借助它和浏览器进行对话

* Node 是一个基于 JavaScript 运行时，它不是一个 JavaScript 框架，也不是一门编程语言。
