#### 导语
> JavaScript 数据类型分为原始类型和引用类型（对象类型）。
<!--more-->               
***

# 原始类型

在 JavaScript 中，存在 6 种原始类型：

* string
* number
* boolean
* undefined
* symbol
* null

原始类型存储的都是值，是没有函数可以调用的。

# 引用类型

引用类型统称为 Object 类型，如果细分的话，常用的有以下几个：

* Object
* Function
* Array
* Date
* Error
* RegExp
